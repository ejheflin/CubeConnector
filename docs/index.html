Finding Your Tenant ID

Follow these steps to locate your Azure AD Tenant ID

  1. Sign in to the Azure Portal
  2. Navigate to Azure Active Directory
  3. Click on Properties in the left menu
  4. Your Tenant ID is displayed under "Directory ID"

The Tenant ID is a GUID in the format: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
